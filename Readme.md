# Business Review Sentiment Analyzer

## Overview
This project analyzes the sentiment of business reviews.
It predicts whether a given review is positive or negative.

## Problem
Businesses receive many customer reviews.
Manually understanding overall sentiment is slow and inconsistent.

## What This Project Does
- Takes a business review as input
- Processes the text
- Predicts whether the sentiment is positive or negative
- Displays the result clearly

## How It Works (High Level)
- User enters a review
- The text is cleaned and prepared
- A trained model generates a sentiment score
- The score is mapped to positive or negative

## Tech Used
- Python
- Streamlit
- Machine Learning model

## Use Case
- Customer feedback analysis
- Review monitoring
- Learning project for text-based sentiment analysis

## Status
Working prototype for learning and demonstration purposes.
